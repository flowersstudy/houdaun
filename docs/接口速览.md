# 接口速览

## 鉴权说明

- 老师端使用 `Authorization: Bearer <teacher_token>`
- 学生端使用 `Authorization: Bearer <student_token>`

## 健康检查

### `GET /health`

- 用途：检查后端服务是否正常
- 返回示例：

```json
{ "status": "ok" }
```

## 认证接口

### `POST /api/auth/teacher/login`

- 用途：老师登录
- 请求体：

```json
{ "email": "li@test.com", "password": "123456" }
```

### `POST /api/auth/student/wx-login`

- 用途：学生微信登录
- 请求体：

```json
{ "code": "wx.login 返回的 code" }
```

### `POST /api/auth/student/bind-phone`

- 用途：学生绑定手机号档案
- 鉴权：学生 token
- 请求体：

```json
{ "phone": "13800000001" }
```

## 学生端接口

### `GET /api/student/profile`

- 用途：获取学生课程进度
- 鉴权：学生 token
- 返回字段：
  - `inProgress`
  - `completed`

### `GET /api/student/leave`

- 用途：获取学生请假记录
- 鉴权：学生 token

### `POST /api/student/leave`

- 用途：提交请假
- 鉴权：学生 token
- 请求体：

```json
{
  "type": "single",
  "courseId": 1,
  "pointName": "游走式找点",
  "stepName": "Day 1 · 1v1共识课",
  "days": 2,
  "reason": "临时有事"
}
```

## 聊天接口

### `GET /api/chat/rooms`

- 用途：获取聊天房间列表
- 鉴权：老师或学生 token

### `GET /api/chat/rooms/:roomId/messages`

- 用途：获取聊天历史
- 鉴权：老师或学生 token

### `WebSocket /?roomId=xxx&token=xxx`

- 用途：实时聊天
- 鉴权：JWT token

## PDF 提交与批改

### `POST /api/submissions`

- 用途：学生上传 PDF 作业/试卷
- 鉴权：学生 token
- 请求方式：`multipart/form-data`
- 文件字段名：`file`
- 额外字段：
  - `studentName`
  - `reviewType`
  - `checkpoint`
  - `deadline`
  - `priority`
  - `submittedNormal`

### `GET /api/submissions`

- 用途：老师查看自己学生的待批改 PDF 列表
- 鉴权：老师 token

### `GET /api/submissions/file/:id`

- 用途：老师读取某条 PDF 文件
- 鉴权：老师 token

### `PUT /api/submissions/:id/grade`

- 用途：老师保存批改结果
- 鉴权：老师 token
- 请求体：

```json
{
  "score": 12,
  "feedback": "结构完整，但论证仍需加强"
}
```

## 通知与请假审批

### `GET /api/student/notifications`

- 用途：获取学生通知列表
- 鉴权：学生 token

### `PATCH /api/student/notifications/read-all`

- 用途：将当前学生全部通知标记为已读
- 鉴权：学生 token

### `PATCH /api/student/notifications/:id/read`

- 用途：将单条学生通知标记为已读
- 鉴权：学生 token

### `GET /api/teacher/tasks/count`

- 用途：获取老师工作台任务计数
- 鉴权：老师 token
- 重点字段：`pendingGrade`、`pendingLeave`、`pendingClass`、`newStudents`、`abnormal`

### `GET /api/teacher/leave-requests`

- 用途：获取当前老师名下待审批请假列表
- 鉴权：老师 token

### `PUT /api/teacher/leave-requests/:id/approve`

- 用途：老师同意学生请假
- 鉴权：老师 token
